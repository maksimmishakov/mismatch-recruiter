================================================================================
  HOW TO RUN
================================================================================

1. Install Dependencies:
   pip install -r requirements.txt

2. Set Environment Variables:
   export YANDEX_GPT_API_KEY="your_key_here"
   export YANDEX_FOLDER_ID="your_folder_id"

3. Start the Server:
   python api_server.py
   # Or: uvicorn api_server:app --reload --host 0.0.0.0 --port 8000

4. Access Application:
   - Web UI: http://localhost:8000/
   - API Docs: http://localhost:8000/docs (interactive)
   - Health: http://localhost:8000/health

5. Test with curl:
   # Phase 1: Extract text
   curl -X POST -F "file=@resume.pdf" http://localhost:8000/api/analyze-resume
   
   # Phase 2: AI analysis
   curl -X POST -F "file=@resume.pdf" http://localhost:8000/api/analyze-resume-ai

## [0.1.0] - 2024-12-25

### Added
- Initial MVP release
- Resume parsing and extraction
- Basic candidate scoring
- Interview question generation
- User authentication (JWT)
- Cloud deployment (Yandex Cloud)
- API documentation
- Comprehensive guides and documentation
- Investor summary
- Technical architecture documentation
- Product roadmap
- FAQ documentation
- Support resources

### Features
- PDF, DOCX, DOC format support
- AI-powered resume analysis
- Automated candidate screening
- Smart interview question generation
- Basic analytics dashboard
- Email notifications
- Slack integration (beta)

### Documentation
- API Reference with all endpoints
- Architecture Guide with system design
- Implementation Guide for Phase 1
- Deployment Guide for various environments
- FAQ covering common questions
- Investor Summary for funding

### Infrastructure
- Yandex Cloud deployment
- PostgreSQL database
- Redis caching layer
- Docker containerization
- Cloud Functions for processing

### Known Limitations
- Single-language support (English)
- Basic analytics (advanced coming in Q1 2025)
- Limited integrations (more coming soon)
- No multi-user collaboration (coming Q1 2025)
